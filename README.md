# Eth-bot
Ai Ethereum chart analysis with future support to other assets such as polygon Celo and Bitcoin
A simple, local dashboard that shows real-time crypto charts and generates automated trading signals using technical indicators like EMA, RSI, MACD, Bollinger Bands, and ATR.

Everything runs on your PC, and data comes directly from Binance through CCXT.
